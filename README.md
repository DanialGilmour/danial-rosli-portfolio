# Danial Rosli - Professional Portfolio

Software Engineer | Data Scientist | Entrepreneur

This is my professional portfolio website, showcasing my experience in Software Engineering, Machine Learning, and my entrepreneurial journey with **MYOSO EMPIRE**.

## ✨ Features
-   **Modern Dark Theme:** Sleek charcoal aesthetic with neon-green/teal accents.
-   **Responsive Design:** Fully optimized for desktop and mobile devices.
-   **Interactive Elements:** Glassmorphism, smooth scroll animations, and parallax effects.
-   **Tech Highlights:** Detailed sections for Machine Learning, DevOps, and Systems Architecture.

## 🚀 How to View
Simply open `index.html` in any modern browser.

## 🌐 Deployment (GitHub Pages)
To publish this on GitHub:
1.  Push this folder to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch and the root directory as the source.
4.  Your portfolio will be live at `https://yourusername.github.io/danial-rosli-portfolio/`.
